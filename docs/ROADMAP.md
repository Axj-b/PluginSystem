# PluginSystem Roadmap

## Later Runtime Features

- Full graph runtime with node scheduling and edge routing.
- POSIX shared-memory backend.
- FIFO buffered ports with overflow policy.
- Streaming ports for long-lived message flows.
- Live rewire support with safe quiescing.
- Cross-process control protocol for remote discovery and triggering.
- Richer node-editor metadata for categories, icons, pin colors, and validation hints.

## Compatibility Work

- ABI compatibility test matrix.
- Descriptor version migration helpers.
- Optional manifest-based discovery cache.
- Plugin package support for resources and dependent DLLs.
